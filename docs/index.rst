========================
EQL Analytics Library
========================

.. image:: _static/eql-whoami.jpg
    :alt: what is eql
    :scale: 50%

**eqllib** is a library of event based analytics, written in `EQL`_ to detect adversary behaviors identified in MITRE `ATT&CKÂ® <https://attack.mitre.org>`_.

.. note::
   Endgame has `joined forces <https://www.elastic.co/blog/endgame-joins-forces-with-elastic>`_ with Elastic, and EQL is now in the `Detection Engine <https://www.elastic.co/guide/en/security/current/rules-ui-create.html#create-eql-rule>`_ of Kibana! To find the latest rules written in EQL, KQL or Lucene for the Elastic Stack, please visit `elastic/detection-rules <https://github.com/elastic/detection-rules>`_ on GitHub.
